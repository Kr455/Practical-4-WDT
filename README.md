# MU Website Replica

## Marwadi University - Department of Computer Engineering

This project is a responsive static web page created for the Department of Computer Science and Engineering at Marwadi University.
It displays important departmental information such as Vision, Mission, Program Specific Outcomes, Courses Offered, and Top Recruiters using HTML and CSS.

---

## Features

* Smooth navigation links to all major sections
* Mobile-friendly responsive layout
* Department Sections:

  * **Vision** – The core purpose of the department
  * **Mission** – Key goals and educational objectives
  * **PSOs (Program Specific Outcomes)** – Learning outcomes for students
  * **Courses Offered** – Various academic programs
  * **Top Recruiters** – Leading hiring companies
* Responsive Design
* Works on desktop, tablet, and mobile devices

---

## Built With

* HTML5
* CSS3
* Google Fonts
